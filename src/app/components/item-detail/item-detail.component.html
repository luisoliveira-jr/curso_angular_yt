Carregando dado individual

Para resgatar um dado do banco vamos precisar criar uma rota dinâmica no nosso router

Baseado no id do item teremos a seleção de dado no banco, geralmente o id vem pela utilizar

Para resgatar parâmetros da URL vamos utilizar o ActivatedRoute, um módulo do Angular
