INTERPOLAÇÃO DE DADOS

É um recurso que vai nos ensinar a trabalhar com os componentes do Angular;

Vamos criar variáveis no arquivo .ts, dentro da classe, ou seja estas variáveis são propriedades da classe e então teremos acesso a estes dados no arquivo .html, o template.

A impressão é feita por meio de um dado entre chaves
