Pipe Operator


Os pipe operators são recursos para trabalhar com strings nos templates

Podemos utilizar diversas funções, como deixar o texto em caixa alta

A sintaxe é dado | algumPipeOperator entre dulpa chaves

Desta maneira nos poupamos de regras de CSS ou manipulação de strings com JS


{{ "Algum texto aqui" | uppercase }}

{{ someText | titlecase }}

{{ today }}

{{ today | date: "fullDate"}} alterando padrão date, com pipe, para fullDate
