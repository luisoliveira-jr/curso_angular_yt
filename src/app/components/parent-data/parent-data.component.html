Compartilhando dados de pai para filho @input:


COMPARTILHANDO DADOS

No Angular podemos compartilhar dados do componente pai para o componente filho

Para isso, vamos disponibilizar na chamada do segundo componente o nome do dado que será recebido com a seguinte sintaxe: dado entre colchetes

No código .ts do componente filho vamos utilizar o decorator @input, que tem como papel entregar o dado para o template.
